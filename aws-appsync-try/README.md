- リゾルバーの実装言語として Apache VTL と JS (APPSYNC_JS) がある

  - APPSYNC_JS では開発環境で eslint とか使えるらしいがベストプラクティスは分からない
  - VTL は API Gateway の影響を受けている気がしている
- ファーストステップとして、GraphQL のスキーマを定義してインポートする必要がある
